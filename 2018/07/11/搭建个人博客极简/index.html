进入HEXO官方文档，可以查看完全的个人博客的搭建。
本文只是对个人在搭建博客过程中，极简的总结，几乎涵盖搭建一个个人免费博客的全部命令，还是非常推荐去官方网站学习。

一、官网主题

在搭建个人博客本地版的时候，总共使用五个命令就好了。

1.npm install hexo-cli -g
2.cd DeskTop
3.hexo init liugezhou
4.npm install
5.hexo s

二、更改主题

三、项目部署

$  hexo clean && hexo g && hexo d
