本周是2020年的第三十二个周末。
这里记录过去一周，我的一些总结。

Feeling

本周看到一篇不用文字叙述而写出来的一篇在当下社会意识形态的大现实，文章不拘一格、行为流水，真乃大作一篇。文章内容记录如下：

张玉环案

张玉环这个案子看的真是让人痛心疾首，那段全家相聚的视频相信很多人看了那叫一个内外流泪。
在我非常浅薄的认知里面，中国司法就是一个笑话。

Mac微信打开小程序

windows版本微信早就可以打开小程序了，而Mac苹果商店版本迟迟没见更新。
在逛微信开放社区的时候，看到原来Mac版本早早就有内测试版本可打开小程序。
安装内测版本不仅可以在微信打开小程序，而且在微信开发者工具调试也可以自己吊起小程序。
Mac小程序开发说明-官方
点击下载

一台服务器支持多个域名和站点

本周在生产服务器通过nginx配置两个域名的时候，花了一整个上午时间总是不成功，原因后来找到原来是对接客户在网关上做了限制，需要网关人员开启域名白名单才可以访问。
这里做个记录：bug的问题不单单是代码原因，还有很多其它外部原因，应多维度去快速定位问题。

Just JavaScript

前几周提到， 亲爱的Dan Abramov–Redux的作者，在编写一本Just JavaScript的草稿书籍。
这本通过邮件订阅方式的专栏现已完结。
书籍比较入门且啰嗦，但挺有意思。
亲爱的Dan一直在强调思维模式养成的重要性，且用几行非常之so easy的代码展开叙述，通篇可能没讲多少内容，但总觉得有一些醍醐灌顶。

绘制变量和值的关系图表

👆 Just JavaScript文章中有很多几行简单的代码，作者希望通过变量与值的关系作图来阐述每一行代码的内部发生了什么。这个网站也是作者推荐的一个绘制变量与值关系图表的在线网站。

10个有用的 git log 技巧

👆– Ten Useful Git Log Tricks.

  • git log --oneline:查看代码提交的一行信息。
  • git log --after='2020-08-01':查看2020年8月1号之后的代码提交记录。
  • git log --after="2020-08-01" --before="2020-08-09":查看2020年8月1日之后，8月9日之前的代码提交记录。
  • git log -p:可以查看每次提交的代码具体修改了那些地方。
  • git log --authot="liumingzhou":查看某位提交人的提交日志。
  • git log --grep="title":查看提交信息带有title字样的提交历史。
