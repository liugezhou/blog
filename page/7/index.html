今天的工作已经做完，在马上就进入五一假期的下班焦躁时刻，来个简单的微信小程序开发规范总结。

1.小程序应避免出现任何 JavaScript 异常

出现 JavaScript 异常可能导致小程序的交互无法进行下去，我们应当追求零异常，保证小程序的高鲁棒性和高可用性，相信这一点一般情况下都不会出现，需要注意的是代码测试中多场景的试错。
